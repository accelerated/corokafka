◆ assignInitialPartitions()

void Bloomberg::corokafka::ConsumerConfiguration::assignInitialPartitions ( PartitionStrategy  strategy,
TopicPartitionList  partitions 
)

Assign partitions and offsets on startup for this consumer.

Parameters
strategyThe strategy to use for this consumer.
partitionsThe partition list.
Remarks
When 'strategy == static', the partitions provided will be used in a call to librdkafka::rd_kafka_assign(). When 'strategy == dynamic', the partition list must contain all partitions for this topic in order to cover any possible partition combinations assigned by Kafka. This setting will result in a call to librdkafka::rd_kafka_subscribe().
